# Supplementary genome archives

Genome FASTA archives for the 586 cyanobacterial genomes shown in Figure 2 are available on Figshare:

- DOI: https://doi.org/10.6084/m9.figshare.32896652
- Figshare item: https://figshare.com/articles/dataset/Genome_archives_for_Figure_2_of_cyanobacterial_nifHDKENB_phylogeny/32896652

Files included in the Figshare dataset:

- `Figure2_286_potential_diazotroph_genomes.zip`: 286 genomes annotated as potential diazotrophs in Supplementary Table S1.
- `Figure2_300_non_diazotroph_genomes.zip`: 300 genomes annotated as non-diazotrophs in Supplementary Table S1.
- `Figure2_586_genomes_manifest.tsv`: manifest linking Supplementary Table S1 rows, assembly IDs, predictions, and FASTA filenames.
- `Figure2_586_genomes_reconciliation.txt`: reconciliation summary.
- `README_Figure2_genome_archives.txt`: brief archive description.

Genome FASTA archives for Supplementary Tables S11 and S12 are available on Figshare:

- DOI: https://doi.org/10.6084/m9.figshare.32816618
- Figshare item: https://figshare.com/articles/dataset/Genome_archives_for_Supplementary_Tables_S11_and_S12_of_cyanobacterial_group_II_nif_genomes/32816618

Files included in the Figshare dataset:

- `S11_genomes.tar.gz`: genomes corresponding to Supplementary Table S11, cyanobacterial genomes carrying group II nif.
- `S12_genomes.tar.gz`: genomes corresponding to Supplementary Table S12, cyanobacterial genomes carrying both group I and group II nif.
